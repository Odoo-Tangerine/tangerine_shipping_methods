Base Vietnam Address

The Base Vietnam Address module is an extension of the Odoo system designed to provide a comprehensive database of addresses in Vietnam. With the integration of the Base Vietnam Address module, Odoo users can access a diverse and detailed collection of addresses at every level, from province/city, district, ward/commune to specific areas. This module has been customized.
